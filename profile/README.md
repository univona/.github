# Univona

**Open-source hybrid messaging platform** — centralized + decentralized + P2P.

## Architecture

- **Daemon** — Lightweight community server core. Deploy one per community.
- **Panel** — Management hub with admin UI. Manages multiple Daemons.
- **Client** — Cross-platform Flutter app with Rust core (E2EE via Signal Protocol).
